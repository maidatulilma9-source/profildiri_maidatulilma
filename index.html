Tentang Saya

Saya Maidatul Ilma Muazama, mahasiswa jurusan sistem komputer di universitas stekom. Saya memiliki minat yang cukup besar pada teknologi, pemograman, dan desain. Seiring berjalannya waktu, saya mulai menyadari bahwa bidang ini bukan hanya tentang komputer dan perangkat keras, tetapi juga tentang rasa penasaran, logika, kreativitas, dan kemampuan memahami bagaimana sesuatu bekerja di balik layar. Setiap materi baru yang saya pelajari selalu membuat saya tertantang untuk memahami lebih dalam, dan dari situlah saya mulai jatuh cinta pada prosesnya.Selain mempelajari sistem komputer dari sisi teknis, saya juga tertarik pada desain, terutama desain web dan tampilan visual. Saya merasa desain bukan hanya soal warna dan bentuk, tetapi tentang bagaimana membuat sebuah tampilan terasa hidup dan mudah dipahami. Saya suka saat bisa menyatukan warna-warna lembut, membuat layout yang rapi, dan menata elemen kecil yang bisa membuat pengguna merasa nyaman. Kadang saya bisa menghabiskan banyak waktu hanya untuk memperbaiki detail kecil sampai akhirnya saya merasa tampilannya benar-benar pas. Dalam hal coding, saya terus mengembangkan kemampuan saya sedikit demi sedikit. Saya belajar HTML, CSS, dan dasar JavaScript, dan bagi saya, coding itu seperti menyusun sebuah puzzle: setiap baris kode punya peran dan ketika semuanya cocok, hasil akhirnya bisa membuat saya tersenyum puas. Meskipun kadang bertemu error atau tampilan yang tidak sesuai keinginan, saya belajar bahwa kesalahan adalah bagian dari proses yang membuat saya lebih memahami apa yang saya lakukan. Justru dari situ saya bisa berkembang.
